    <p class="line867">
     <h2 id="Welcome">
      Welcome
     </h2>
     <span class="anchor" id="line-2">
     </span>
     <p class="line874">
      ID3 tags provide the Title, Artist, Year, Genre and other great information when you're listening to music. Here you will find the current standards documents; pointers to software libraries in just about every language and other information you can use to enhance your MP3 audio library.
      <span class="anchor" id="line-3">
      </span>
      <span class="anchor" id="line-4">
      </span>
      <p class="line862">
       ID3 tags are the audio file data standard for MP3 files in active use by software and hardware developers around the world.  ID3 tags are supported in software such as
       <a class="http" href="http://itunes.com">
        iTunes
       </a>
       ,
       <a class="http" href="http://www.microsoft.com/windows/windowsmedia/">
        Windows Media Player
       </a>
       ,
       <a class="http" href="http://winamp.com/">
        Winamp
       </a>
       ,
       <a class="http" href="http://www.videolan.org/">
        VLC
       </a>
       , and hardware players like the
       <a class="http" href="http://www.apple.com/ipod">
        iPod
       </a>
       ,
       <a class="http" href="http://us.creative.com/p/mp3player">
        Creative Zen
       </a>
       ,
       <a class="http" href="http://www.samsung.com/us/mobile/mp3-players">
        Samsung Galaxy
       </a>
       , and
       <a class="http" href="http://store.sony.com/walkman-mp3-players/cat-27-catid-All-MP3-Players">
        Sony Walkman
       </a>
       .
       <span class="anchor" id="line-5">
       </span>
       <span class="anchor" id="line-6">
       </span>
       <p class="line862">
        An ID3 tag is a data container within an MP3 audio file stored in a prescribed format. This data commonly contains the Artist name, Song title, Year and Genre of the current audio file.  This website contains the format standards information for the ID3 tagging data container.  If you've read this far and are confused, check the
        <a href="/ID3v2Easy">
         ID3v2Easy
        </a>
        page for a short, low-tech introduction.
        <span class="anchor" id="line-7">
        </span>
        <span class="anchor" id="line-8">
        </span>
        <p class="line862">
         While there are legacy and future standards for ID3 tags, the most popular version implemented today is ID3
         <a href="/id3v2.3.0">
          version 2.3
         </a>
         .  A follow on version,
         <a href="/id3v2.4.0-structure">
          2.4
         </a>
         , is
         <span class="anchor" id="line-9">
         </span>
         documented on this website but has not achieved popular status due to some disagreements on some of the revisions and the tremendous inertia present in the software and hardware marketplace.
         <span class="anchor" id="line-10">
         </span>
         <span class="anchor" id="line-11">
         </span>
         <p class="line862">
          Consumers should read the
          <a href="/Introduction">
           Introduction
          </a>
          where the basics of the ID3 tagging format and its history are covered.  On that page are links to basic technical information about how ID3 tags work. The
          <a href="/FAQ">
           Frequently Asked Questions (FAQ)
          </a>
          page contains some additional items of interest.
          <span class="anchor" id="line-12">
          </span>
          <span class="anchor" id="line-13">
          </span>
          <p class="line862">
           Developers who are new to the ID3 tag format may want to jump directly to the
           <a href="/ID3v2Easy">
            ID3v2 made easy
           </a>
           page for quick low tech introduction. Those interested in getting right into the thick of the standard should check out the
           <a href="/Developer%20Information">
            Developer Information
           </a>
           page. Software developers will find ID3 tag manipulation libraries under
           <a href="/Implementations">
            Implementations
           </a>
           .
           <span class="anchor" id="line-14">
           </span>
           <span class="anchor" id="line-15">
           </span>
           <p class="line862">
            The technical aspects of ID3 tagging are discussed on the
            <a href="/MailingList">
             ID3v2 Developers Mailing List
            </a>
            .
